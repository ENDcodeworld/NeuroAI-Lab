## 3. 获客策略

### 3.1 B2C 获客（神经反馈 App）

#### 渠道矩阵

| 渠道 | 预算占比 | CAC 目标 | 转化周期 |
|------|---------|---------|---------|
| 内容营销 | 30% | ¥50 | 7-14 天 |
| KOL 合作 | 25% | ¥80 | 3-7 天 |
| 应用商店 | 15% | ¥30 | 1-3 天 |
| 社交媒体广告 | 20% | ¥100 | 3-7 天 |
| 企业团购 | 10% | ¥20 | 30-60 天 |

#### 内容营销策略

**平台选择：**
- 知乎：专业内容，建立权威
- 小红书：用户体验分享
- B 站：科普视频 + 产品演示
- 微信公众号：深度文章 + 私域运营

**内容类型：**
```
1. 科普内容（引流）
   - "脑电波是什么？3 分钟看懂"
   - "专注力可以训练吗？科学告诉你"
   - 目标：建立认知，获取潜在用户

2. 产品内容（转化）
   - "我用 NeuroAI 训练专注力 30 天，结果..."
   - "神经反馈训练真实体验"
   - 目标：展示效果，促进购买

3. 用户故事（信任）
   - 用户案例采访
   - 前后对比数据
   - 目标：社会证明，增强信任
```

**KOL 合作策略：**
- 头部 KOL（100 万 + 粉丝）：品牌曝光，¥5-10 万/条
- 腰部 KOL（10-50 万粉丝）：效果转化，¥1-3 万/条
- 微型 KOL（1-10 万粉丝）：口碑传播，¥1000-5000/条 + 佣金

#### 转化漏斗设计

```
认知 → 兴趣 → 试用 → 付费 → 留存

1. 认知层（曝光）
   - 内容阅读/视频观看
   - 目标：10 万曝光/月

2. 兴趣层（访问）
   - 官网/App Store 访问
   - 目标：1 万访问/月（10% 点击率）

3. 试用层（下载）
   - App 下载注册
   - 目标：3000 下载/月（30% 转化）

4. 付费层（订阅）
   - 免费试用→付费
   - 目标：450 付费/月（15% 转化）

5. 留存层（续费）
   - 月续费率
   - 目标：70% 月留存
```

## 5. 融资规划

### 5.1 融资节奏

```
种子轮 → 天使轮 → Pre-A 轮 → A 轮

种子轮（现在）
├── 金额：300-500 万
├── 出让：10-15%
├── 用途：产品开发（60%）、团队（30%）、运营（10%）
└── 里程碑：MVP 上线，100 付费用户

天使轮（M6-M9）
├── 金额：1000-1500 万
├── 出让：10-15%
├── 用途：市场推广（50%）、团队扩张（30%）、产品迭代（20%）
└── 里程碑：1000 付费用户，月收入 50 万

Pre-A 轮（M12-M15）
├── 金额：3000-5000 万
├── 出让：10-15%
├── 用途：规模化扩张
└── 里程碑：月收入 150 万，盈亏平衡

A 轮（M18-M24）
├── 金额：1-2 亿
├── 出让：10-15%
├── 用途：市场占领、并购、国际化
└── 里程碑：月收入 500 万+，行业领先
```

### 5.2 投资人画像

**种子轮：**
- 个人天使（成功创业者）
- 早期 VC（真格、险峰、英诺）
- 产业投资人（医疗健康基金）

**天使轮及以后：**
- 专业 VC（红杉、IDG、经纬）
- 医疗科技基金（启明、礼来亚洲）
- 战略投资人（医疗设备公司）

### 5.3 估值逻辑

```
估值方法：PS（市销率）法

参考可比公司：
- Muse: PS 8-10x
- Emotiv: PS 6-8x
- 国内数字健康：PS 5-8x

我们的估值：
M6（月收入 50 万）：年收入 600 万 × PS 8x = 4800 万
M12（月收入 100 万）：年收入 1200 万 × PS 10x = 1.2 亿
M24（月收入 250 万）：年收入 3000 万 × PS 12x = 3.6 亿
```

## 6. 风险管理

### 6.1 收入风险

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| 用户增长不及预期 | 中 | 高 | 多元化获客渠道、快速迭代 |
| 付费转化率低 | 中 | 高 | A/B 测试、优化定价、增加价值 |
| 用户流失率高 | 中 | 高 | 提升产品粘性、建立社区 |
| B2B 销售周期长 | 高 | 中 | 提前储备线索、优化销售流程 |

### 6.2 成本风险

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| 人力成本上涨 | 中 | 中 | 远程办公、外包非核心 |
| 云服务成本超支 | 低 | 中 | 优化架构、预留 buffer |
| 营销 ROI 低 | 中 | 高 | 数据驱动、快速止损 |

### 6.3 应对预案

```
预案 A（增长不及预期 50%）：
- 缩减营销预算 30%
- 聚焦高 ROI 渠道
- 延长 runway 至 18 个月

预案 B（收入停滞 3 个月）：
- 启动紧急融资
- 裁员 20-30%（非核心）
-  pivot 到 B2B 为主

预案 C（现金流危机）：
- 创始人降薪/无薪
- 应收账款保理
- 过桥贷款
```
